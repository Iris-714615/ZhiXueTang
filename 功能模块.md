# 知乎知学堂 · 三端功能模块梳理

## 整体三端架构

```
┌─────────────────────────────────────────────────────────┐
│                    知乎知学堂三端架构                      │
├──────────────┬──────────────────┬───────────────────────┤
│  学员端 (C端) │  机构端/讲师端(B端)│   管理端 (Admin端)    │
│  vue-pro SPA │  创作中心+视频上传 │  Django Admin 后台    │
└──────┬───────┴────────┬─────────┴──────────┬────────────┘
       │                │                    │
       └────────────────┼────────────────────┘
                        ▼
        Django 8000 (业务) + FastAPI 8001 (AI)
        + MySQL + Redis + ES + Celery
```

---

## 一、学员端（C端 / 用户端）

> 面向学员用户，核心学习与消费场景。前端路由 16 个页面。

### 1.1 用户认证模块
| 功能 | 技术实现 | 对应代码 |
| :--- | :--- | :--- |
| 用户注册 | 手机号+短信验证码注册 | Register.vue |
| 用户登录 | JWT 签发（user_id/username/exp/iat） | Login.vue |
| 密码重置 | 邮箱验证码重置 | ResetPassword.vue |
| 登录状态管理 | Pinia userStore 全局响应式 | store/userStore.ts |
| 鉴权路由守卫 | localStorage token + router.beforeEach | router/index.js |

### 1.2 课程浏览模块
| 功能 | 技术实现 | 对应代码 |
| :--- | :--- | :--- |
| 首页展示 | Banner轮播 + 分类导航 + 课程推荐 | Home.vue |
| 课程列表 | 分类筛选 + 分页 + 标签过滤 | Courselist.vue |
| 课程详情 | 课程介绍/章节课时/讲师/评价/试听 | CourseDetail.vue |
| 课程搜索 | Elasticsearch 全文检索 | Search.vue |

### 1.3 购物与支付模块
| 功能 | 技术实现 | 对应代码 |
| :--- | :--- | :--- |
| 购物车 | 添加/删除/批量结算 | Cart.vue |
| 结算页 | 优惠券抵扣 + 金额计算 | Checkout.vue |
| 微信支付 | 扫码支付 + 支付回调 | WxPayModal.vue |
| 支付宝支付 | 网页支付 + 异步回调 | tools/pay.py |
| 支付成功 | 订单状态回写 + 权益开通 | PaySuccess.vue |

### 1.4 学习中心模块
| 功能 | 技术实现 | 对应代码 |
| :--- | :--- | :--- |
| 我的课程 | 已购课程列表 + 有效期 | CourseStudy.vue |
| 视频学习 | 视频播放 + 进度记录 | CourseStudyEnhanced.vue |
| 学习进度 | 进度异步写入（守护线程） | euser/ai_views.py |
| 课程评价 | 星级 + 文字评价 | User.vue |
| 个人中心 | 用户信息/余额/积分/订单 | User.vue |

### 1.5 AI 伴学助手模块（核心亮点）
| 功能 | 技术实现 | 对应代码 |
| :--- | :--- | :--- |
| SSE 流式对话 | fetch ReadableStream + StreamingResponse | services/aiService.ts |
| 意图识别 | LangGraph 状态图（rag/chat/code 三路由） | services/llm_service.py |
| RAG 知识问答 | LlamaIndex 双索引 + ES 向量检索 | services/rag_service.py |
| 混合检索增强 | 多路召回 + 归一化 + BGE-reranker | retrieval/hybrid_retriever.py |
| Markdown 渲染 | marked 库实时解析 | components/AIChatWindow.vue |

### 1.6 直播间互动模块
| 功能 | 技术实现 | 对应代码 |
| :--- | :--- | :--- |
| 实时弹幕 | AsyncWebsocketConsumer + group_send | live/consumers.py |
| 在线人数 | Redis INCR/DECR 原子计数 | live/consumers.py |
| 心跳保活 | 服务端30s ping + 客户端25s ping | services/websocketService.ts |
| 断线重连 | 指数退避（1/2/4/8/16s）+ 历史弹幕补全 | LiveRoom.vue |
| 弹幕限流 | Token Bucket（3条/秒/用户） | live/consumers.py |
| 弹幕持久化 | Redis Streams XADD（保留1000条/24h） | live/consumers.py |

## 五、三端职责对比

| 维度 | 学员端 (C端) | 机构端 (B端) | 管理端 (Admin) |
| :--- | :--- | :--- | :--- |
| **用户** | 学员 | 讲师/机构 | 平台管理员 |
| **入口** | vue-pro SPA (5173) | Header创作中心+视频上传 | Django Admin (/admin) |
| **核心职责** | 学习消费 | 内容生产 | 运营审核 |
| **页面数** | 16个 | 视频上传+创作中心 | Admin后台 |
| **鉴权** | JWT + Pinia | JWT + 权限校验 | Django Session + Staff |
| **关键技术** | Vue3+Pinia+SSE+WS | 分片上传+Celery+FFmpeg | Django Admin + ORM |
| **数据写入** | 订单/评价/进度 | 课程/视频/知识库 | 分类/Banner/审核 |

## 六、技术栈总览

| 层级 | 技术栈 |
| :--- | :--- |
| **前端** | Vue 3 + TypeScript + Pinia + Vue Router + Element Plus + Axios + Vite |
| **核心业务后端** | Django 5.2 + DRF + Django Channels + Daphne (ASGI) |
| **AI 微服务** | FastAPI + Uvicorn + Pydantic |
| **AI 编排与 RAG** | LangGraph（状态图 Agent）+ LlamaIndex（双索引 RAG）+ LangChain |
| **大语言模型** | 千问 qwen-plus（DashScope OpenAI 兼容接口） |
| **数据存储** | MySQL 8.0 + Redis 7.0 + Elasticsearch 8.x（向量检索） |
| **异步任务** | Celery + Redis Broker |
| **实时通信** | WebSocket（Django Channels）+ SSE（FastAPI StreamingResponse） |
| **视频处理** | FFmpeg（转码）+ Whisper（语音识别字幕提取） |
| **工程化** | JWT 双服务共享密钥鉴权 + CORS 预检 + 工厂模式 + 单例模式 |

> 三端共用同一套 Django + FastAPI 后端服务，通过 JWT 统一鉴权，MySQL 作为共享主库，实现数据互通。
