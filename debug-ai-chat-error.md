## 假设（待证据验证）
1. **H1 - JWT 鉴权失败**：前端 token 传递格式错误或 Django 签发 token 与 FastAPI 验证密钥不一致
2. **H2 - SSE 流式响应被中间件拦截**：JWT 中间件或 CORS 中间件在流式响应过程中提前关闭连接
3. **H3 - 前端请求路径/方法不匹配**：前端调用路径与 FastAPI 路由定义不一致
4. **H4 - LLM/RAG 服务内部异常**：DashScope API 调用失败、索引未构建、embedding 模型加载失败
5. **H5 - 用户上下文获取失败**：从 token 解析 user_id 后查 Django 用户库失败，导致上下文缺失
